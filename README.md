# e-plantShopping

Paradise Nursery shopping cart project built with **React + Redux Toolkit**.

## Repository

This repository is named **e-plantShopping**.

## Features

- Landing page with background image and a **Get Started** button
- Product listing grouped into categories with **Add to Cart** buttons
- Cart icon badge shows the **total quantity** in the cart
- Shopping cart page:
  - Update quantities (+ / -)
  - Remove items
  - Per-item subtotal and overall total
  - Checkout button shows a “coming soon” style message

## Run locally

```bash
npm install
npm run dev
```
